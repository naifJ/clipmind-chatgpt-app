Smart PDF Assistant Privacy Policy

Last updated: May 16, 2026

Smart PDF Assistant is a ChatGPT app that processes user-provided PDF files for operations such as merging, splitting, and invoice data extraction.

Information We Process

The app processes files you intentionally provide, file names, file metadata, requested operation settings, and generated output files. The app does not require account credentials, payment details, or OAuth login.

Temporary Storage

Output files are stored temporarily so you can download them. Temporary files are configured to expire automatically. We do not intentionally keep a permanent database of uploaded PDF content.

External Services

The current MVP performs processing on the app server and does not send files to external OCR or AI services. If external OCR or AI processing is enabled later, it should be disclosed in configuration and policy updates.

Security Limits

  • PDF file type validation is performed before PDF operations.
  • Executable files are not accepted for PDF tools.
  • File size and request limits are enforced.
  • Operational logs avoid storing file content.

User Choices

Do not submit confidential, sensitive, medical, legal, financial, or personal documents unless you are comfortable having them processed for the requested operation.

سياسة الخصوصية بالعربية

Smart PDF Assistant يعالج ملفات PDF التي يرسلها المستخدم لتنفيذ عمليات مثل الدمج، التقسيم، واستخراج بيانات الفواتير.

لا يحتاج التطبيق إلى تسجيل دخول أو OAuth. يتم حفظ الملفات الناتجة مؤقتاً فقط حتى يستطيع المستخدم تنزيلها، ثم تنتهي حسب إعدادات الحذف المؤقت.

النسخة الحالية لا ترسل الملفات إلى خدمات OCR أو ذكاء اصطناعي خارجية. إذا تم تفعيل ذلك لاحقاً يجب توضيحه في الإعدادات والسياسة.

لا ترسل مستندات سرية أو شديدة الحساسية إلا إذا كنت مرتاحاً لمعالجتها لتنفيذ العملية المطلوبة.
